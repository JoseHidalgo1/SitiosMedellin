# Medellín Movilidad | Práctica Académica

¡Bienvenido al sitio web informativo de los lugares emblemáticos visitados en la práctica académica en Rionegro y el Área Metropolitana de Medellín!

## Descripción del Proyecto

Este proyecto presenta los principales sistemas de movilidad, transporte y espacios emblemáticos de la región, integrando datos curiosos, visuales, cifras urbanas y secciones interactivas para una experiencia moderna e instructiva. El objetivo es que el usuario (profesor o estudiante) conozca, de forma clara y atractiva, los lugares clave de la práctica académica universitaria.

## Funcionalidades

- Grid interactivo con 8 lugares emblemáticos
- Modales emergentes con mucha información y fotos reales por cada lugar
- Sección de historia de transformación urbana (timeline)
- Apartado “Medellín: Ciudad de la Eterna Primavera” con datos de clima, geografía, cultura y reconocimientos
- Competencias que aprenderás en la práctica (cards)
- Datos y cifras importantes del sistema de transporte
- Hechos curiosos visuales
- Diseño responsivo y profesional

## Instalación y publicación

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador para vista local.

## Tecnologías utilizadas

- HTML5, CSS3, JavaScript puro
- Framework de íconos: Font Awesome CDN
- Recursos visuales libres/creados para la academia
